# أتمتة الردود على قصص إنستغرام

تعد أتمتة الردود على قصص إنستغرام باستخدام Instagram Graph API طريقة فعالة للتفاعل مع جمهورك. ومع ذلك، يجب على المطورين التعامل مع قيود صارمة لواجهة برمجة التطبيقات، خاصة فيما يتعلق بنوافذ المراسلة والوصول إلى الوسائط.

## قيود واجهة المستخدم وقيود الـ 24 ساعة

القيود الأكثر أهمية عند التعامل مع الردود على القصص هي **نافذة المراسلة التي تبلغ 24 ساعة**.

- **نافذة المراسلة:** عندما يرد مستخدم على قصتك في إنستغرام، فإنه يبدأ محادثة، مما يفتح نافذة مراسلة مدتها 24 ساعة. يمكنك إرسال رسائل آلية فقط خلال هذه الفترة.
- **انتهاء صلاحية الوسائط:** القصص مؤقتة بطبيعتها. يمكن الوصول إلى كائن `IG Media` الذي يمثل القصة عبر واجهة برمجة التطبيقات لمدة 24 ساعة فقط. بعد ذلك، ستفشل أي محاولة لجلب وسائط القصة أو الرؤى الخاصة بها.

<!-- preview -->
✅ Valid: إرسال رسالة ترحيب آلية فور رد المستخدم على قصتك.
❌ Invalid: محاولة إرسال رسالة ترويجية آلية بعد 48 ساعة من آخر تفاعل للمستخدم.

## قيود جلب الوسائط

عندما تتلقى إشعار Webhook لرد على قصة، تتضمن البيانات إشارة إلى وسائط القصة. ومع ذلك، هناك قيود:
- إذا كنت بحاجة إلى تحليل الوسائط (مثل تنزيل الصورة أو الفيديو)، فيجب عليك القيام بذلك خلال نافذة الـ 24 ساعة.
- لن تُرجع واجهة برمجة التطبيقات بيانات للقصص منتهية الصلاحية، حتى لو كان لديك معرّف الوسائط (Media ID).

## هيكل الاستجابة (API Payload)

عادةً ما تحتوي بيانات Webhook لرد القصة على كائن `story` متداخل داخل بيانات الرسالة.

```json
{
  "object": "instagram",
  "entry": [
    {
      "id": "<IG_ACCOUNT_ID>",
      "time": 1600000000,
      "messaging": [
        {
          "sender": { "id": "<IGSID>" },
          "recipient": { "id": "<IG_ACCOUNT_ID>" },
          "timestamp": 1600000000,
          "message": {
            "mid": "<MESSAGE_ID>",
            "text": "Love this!",
            "reply_to": {
              "mid": "<STORY_MESSAGE_ID>",
              "story": {
                "url": "<STORY_MEDIA_URL>",
                "id": "<STORY_MEDIA_ID>"
              }
            }
          }
        }
      ]
    }
  ]
}
```
<!-- preview -->
✅ Valid: استخراج `story.url` في الدقائق الأولى لتسجيل التفاعل.
❌ Invalid: محاولة الاستعلام عن واجهة برمجة التطبيقات للحصول على `<STORY_MEDIA_ID>` بعد يومين.\n\n
<!-- panel:comparison -->
**الإشارات في القصة (Mentions) مقابل الردود على القصة (Replies)**
- **الإشارة (Mentions):** تحدث عندما يذكرك مستخدم في قصته الخاصة (Tag). تتلقى إشعار Webhook بذلك.
- **الردود (Replies):** تحدث عندما يرفع المستخدم الشاشة للرد على قصتك. تُعامل كرسالة عادية.
