# واجهة برمجة تطبيقات الرسائل (إرسال الرسائل)

تُعد واجهة برمجة تطبيقات الرسائل (Messages API) الطريقة التي يرسل بها نشاطك التجاري الرسائل إلى المستخدمين — النصوص، والوسائط، والقوالب، والرسائل التفاعلية، والمزيد.

## نقطة النهاية (Endpoint)

```http
POST /v21.0/{phone-number-id}/messages
```

---

## المعلمات الأساسية (جميع الطلبات)

| الحقل | النوع | مطلوب | القيم الصالحة |
|-------|------|----------|-------------|
| `messaging_product` | string | ✅ | `"whatsapp"` |
| `recipient_type` | string | ❌ | `"individual"` (الافتراضي) أو `"group"` |
| `to` | string | ✅ | رقم الهاتف مع رمز الدولة (بدون `+`) أو معرّف المجموعة (group ID) |
| `type` | string | ✅ | `text`, `image`, `video`, `audio`, `document`, `sticker`, `location`, `contacts`, `interactive`, `template`, `reaction` |
| `context` | object | ❌ | `{ "message_id": "wamid.xxx" }` — الاقتباسات/الردود |
| `biz_opaque_callback_data` | string | ❌ | بحد أقصى 512 حرفًا — يتم إرجاعها في خطاف الويب للحالة (status webhook) |

## أنواع الرسائل — المعلمات والتحقق من الصحة

### TEXT (نص)

```json
{
  "messaging_product": "whatsapp",
  "to": "15551234567",
  "type": "text",
  "text": {
    "body": "Hello, how can I help you?",
    "preview_url": false
  }
}
```

| الحقل | الحد | ملاحظات |
|-------|-------|-------|
| `text.body` | بحد أقصى **4096** حرفًا | يدعم *المائل*، **العريض**، ~~يتوسطه خط~~، `أحادي المسافة` |
| `text.preview_url` | boolean | `true` = عرض معاينة الرابط |

### قواعد التحقق من صحة النص (TEXT)

| القاعدة | ❌ مرفوض | ✅ مقبول |
|------|------------|------------|
| طول النص | > 4096 حرفًا | ≤ 4096 حرفًا |
| الإرسال لغير العملاء | بمبادرة من النشاط التجاري بدون نافذة مفتوحة وبدون قالب | إرسال قالب بدلاً من ذلك |
| علامات HTML | `<b>text</b>` | `*text*` (تنسيق ماركداون لـ WhatsApp) |

---

### IMAGE / VIDEO / AUDIO / DOCUMENT (صورة / فيديو / صوت / مستند)

يمكن إرسال الوسائط باستخدام `id` (مرفوعة إلى Meta) أو `link` (رابط مباشر):

```json
// ✅ By URL
{
  "type": "image",
  "image": {
    "link": "https://example.com/image.jpg",
    "caption": "This is a caption"
  }
}

// ✅ By ID (pre-uploaded)
{
  "type": "image",
  "image": {
    "id": "MEDIA_ID",
    "caption": "Caption here"
  }
}
```

### حدود الوسائط

| النوع | الحد الأقصى للحجم | التنسيقات المقبولة |
|------|---------|-----------------|
| صورة (Image) | **5 ميغابايت** | JPEG, PNG |
| فيديو (Video) | **16 ميغابايت** | MP4, 3GPP |
| صوت (Audio) | **16 ميغابايت** | AAC, MP4, MPEG, AMR, OGG |
| مستند (Document) | **100 ميغابايت** | PDF، Word، Excel، PowerPoint، text |
| ملصق (Sticker) | **500 كيلوبايت** | WebP (ثابت)، WebP متحرك |

### قواعد التحقق من صحة الوسائط

| القاعدة | ❌ مرفوض | ✅ مقبول |
|------|------------|------------|
| تنسيق الصورة | GIF، BMP، TIFF | JPEG، PNG |
| تنسيق الفيديو | AVI، MOV، WMV | MP4، 3GPP |
| الوسائط بالرابط | رابط خاص/يتطلب المصادقة | رابط HTTPS يمكن الوصول إليه للعامة |
| التسمية التوضيحية للصوت | إضافة `caption` للصوت | الصوت لا يدعم التسميات التوضيحية |
| التسمية التوضيحية للملصق | إضافة `caption` للملصق | الملصق لا يدعم التسميات التوضيحية |
| `id` و `link` معًا | إدراج كلا الحقلين في نفس الكائن | استخدم أحدهما |

### التفاعلية (INTERACTIVE) — رسائل القوائم

```json
{
  "type": "interactive",
  "interactive": {
    "type": "list",
    "header": { "type": "text", "text": "Choose an option" },
    "body": { "text": "Select from the list below:" },
    "footer": { "text": "Tap to choose" },
    "action": {
      "button": "View Options",
      "sections": [
        {
          "title": "Support",
          "rows": [
            { "id": "row_1", "title": "Billing Issue", "description": "Help with billing" },
            { "id": "row_2", "title": "Tech Support", "description": "Technical help" }
          ]
        }
      ]
    }
  }
}
```

### حدود القائمة التفاعلية

| العنصر | الحد |
|---------|-------|
| الأقسام (Sections) | بحد أقصى **10** |
| الصفوف لكل قسم | بحد أقصى **10** |
| إجمالي الصفوف | بحد أقصى **10** |
| نص الزر | بحد أقصى **20** حرفًا |
| عنوان القسم | بحد أقصى **24** حرفًا |
| عنوان الصف | بحد أقصى **24** حرفًا |
| وصف الصف | بحد أقصى **72** حرفًا |
| معرّف الصف (Row ID) | بحد أقصى **200** حرف |
| نص الرأس (Header) | بحد أقصى **60** حرفًا |
| نص المتن (Body) | بحد أقصى **4096** حرفًا |
| نص التذييل (Footer) | بحد أقصى **60** حرفًا |

### التفاعلية (INTERACTIVE) — أزرار الرد

```json
{
  "type": "interactive",
  "interactive": {
    "type": "button",
    "body": { "text": "Choose an action:" },
    "action": {
      "buttons": [
        { "type": "reply", "reply": { "id": "btn_yes", "title": "Yes" } },
        { "type": "reply", "reply": { "id": "btn_no",  "title": "No" } }
      ]
    }
  }
}
```

| العنصر | الحد |
|---------|-------|
| الحد الأقصى للأزرار | **3** |
| عنوان الزر | بحد أقصى **20** حرفًا |
| معرّف الزر (Button ID) | بحد أقصى **256** حرفًا، ويجب أن يكون فريدًا |

### القوالب (TEMPLATE)

```json
{
  "type": "template",
  "template": {
    "name": "order_shipped",
    "language": { "code": "en_US" },
    "components": [
      {
        "type": "header",
        "parameters": [
          { "type": "image", "image": { "link": "https://example.com/img.jpg" } }
        ]
      },
      {
        "type": "body",
        "parameters": [
          { "type": "text", "text": "Sarah" },
          { "type": "text", "text": "#ORD-1234" }
        ]
      },
      {
        "type": "button",
        "sub_type": "url",
        "index": "0",
        "parameters": [
          { "type": "text", "text": "ABC123" }
        ]
      }
    ]
  }
}
```

### التحقق من صحة إرسال القالب

| القاعدة | ❌ مرفوض | ✅ مقبول |
|------|------------|------------|
| عدد المتغيرات | عدد خاطئ من الـ `parameters` | عدد دقيق يطابق تعريف القالب |
| رمز اللغة | `"en"` لقالب `en_US` | `"en_US"` بالضبط |
| حالة القالب | إرسال قالب بحالة `REJECTED` أو `PAUSED` | القوالب بحالة `APPROVED` فقط |
| نوع المتغير | معلمة `text` لرأس الصورة | معلمة `image` أو `video` لرأس الوسائط |
| فهرس الزر | `index` خاطئ لترتيب الزر | `"0"` للأول، `"1"` للزر الثاني |
| معلمة فارغة | `"text": ""` | سلسلة نصية غير فارغة |

---

### التفاعل (REACTION)

```json
{
  "type": "reaction",
  "reaction": {
    "message_id": "wamid.xxx",
    "emoji": "❤️"
  }
}
```

| القاعدة | ❌ مرفوض | ✅ مقبول |
|------|------------|------------|
| الرمز التعبيري (Emoji) | نص ليس رمزًا تعبيريًا، عدة رموز تعبيرية | رمز تعبيري واحد فقط |
| إزالة التفاعل | حذف `emoji` | `"emoji": ""` |
| الرسالة المستهدفة | رسالة من حساب واتساب للأعمال (WABA) آخر | رسالة في نفس المحادثة |

## نوافذ المراسلة (متى يمكنك الإرسال)

| نوع النافذة | المدة | ما يمكنك إرساله |
|-------------|----------|-------------------|
| **خدمة العملاء** | 24 ساعة من آخر رسالة للمستخدم | أي نوع من الرسائل |
| **نقطة دخول مجانية** | 72 ساعة من إعلان النقر للوصول إلى WhatsApp | أي نوع من الرسائل (مجانًا) |
| **خارج النافذة** | بعد 24 ساعة من عدم الاستجابة | القوالب فقط |

## حدود المعدل

| الحد | القيمة |
|-------|-------|
| الرسائل في الثانية (لكل رقم هاتف) | **80 رسالة/ثانية** |
| رسائل القوالب لنفس المستخدم | **القوالب التسويقية**: حد لكل مستخدم يُفرض بواسطة Meta |
| فئة رقم الهاتف | تتسع مع الاستخدام: 1 ألف ← 10 آلاف ← 100 ألف ← غير محدود/يوم |

## الاستجابة (النجاح)

```json
{
  "messaging_product": "whatsapp",
  "contacts": [{ "input": "15551234567", "wa_id": "15551234567" }],
  "messages": [{ "id": "wamid.HBgNMTU1NTEyMzQ1Njc..." }]
}
```

احفظ `messages[0].id` — ستحتاج إليه لربطه بخطافات الويب (webhooks) الخاصة بالحالة.

## مرجع رموز الخطأ

| الرمز | الخطأ | السبب | الإصلاح |
|------|-------|-------|-----|
| `100` | معلمة غير صالحة | قيمة أو تنسيق حقل خاطئ | تحقق من هيكل البيانات المرسلة (payload) |
| `131000` | خطأ عام | أسباب مختلفة | تحقق من رسالة الخطأ الكاملة |
| `131005` | تم رفض الإذن | الرمز المميز (Token) يفتقر إلى الإذن | منح إذن `whatsapp_business_messaging` |
| `131008` | معلمة مطلوبة مفقودة | غياب `to` أو `type` أو ما إلى ذلك | إضافة الحقل المفقود |
| `131009` | قيمة المعلمة غير صالحة | هاتف غير صالح، وسائط غير صالحة | تحقق من صحة المدخلات |
| `131021` | مستلم غير صالح | رقم الهاتف غير موجود على WhatsApp | تحقق من الرقم |
| `131026` | تعذر تسليم الرسالة | لا يمكن الوصول إلى المستخدم | لا يوجد إجراء — قم بالتسجيل وانتقل للخطوة التالية |
| `131047` | إعادة التفاعل مطلوبة | لا توجد نافذة مفتوحة | إرسال قالب |
| `131048` | حد معدل البريد العشوائي (Spam) | إرسال رسائل أكثر من اللازم | تراجع وقلل من الإرسال |
| `131049` | حد التسويق | وصل المستخدم إلى الحد الأقصى | التزم بالحد الأقصى |
| `131050` | المستخدم ألغى الاشتراك | ألغى المستخدم اشتراكه | احترم إلغاء الاشتراك |
| `131051` | نوع غير مدعوم | لا يمكن إرسال هذا النوع | استخدم نوعًا مختلفًا |
| `130429` | تم الوصول إلى حد المعدل | الإرسال بسرعة كبيرة | قلل الإرسال إلى 80/ثانية |
| `132000` | عدد معلمات القالب خاطئ | عدد المتغيرات خاطئ | أصلح مصفوفة المعلمات |
| `132001` | القالب غير موجود/غير معتمد | اسم/لغة خاطئة | استخدم قالبًا معتمدًا |

## ملاحظات SEEN V2

| الميزة | ملاحظات |
|---------|-------|
| البث (Broadcasts) | استخدم نوع `template`. قم دائمًا بتمرير العدد الدقيق للمعلمات. |
| ردود العملاء | تعامل مع خطاف الويب `messages`، وقم بتحليلها حسب النوع `type` |
| الرد/السياق | قم بتعيين `context.message_id` لاقتباس رسالة المستخدم |
| رسائل الوسائط | قم بتخزين `media.id` من خطاف الويب — لا تقم بإعادة تنزيلها من الرابط (لأنه تنتهي صلاحيته) |
| تتبع الحالة | قم بتخزين `messages[0].id` وتتبع الحالات في خطاف الويب |\n\n

**الرسائل النصية مقابل رسائل القوالب**
- **الرسائل النصية الحرة:** يمكن إرسالها فقط إذا كانت نافذة خدمة العملاء (24 ساعة) مفتوحة.
- **رسائل القوالب (Templates):** يمكن إرسالها في أي وقت لبدء محادثة تجارية، حتى خارج نافذة الـ 24 ساعة.
