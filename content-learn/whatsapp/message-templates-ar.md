# قوالب الرسائل

القوالب هي تنسيقات رسائل معتمدة مسبقاً ومطلوبة للمحادثات التي تبدأ من قبل النشاط التجاري خارج نافذة خدمة العملاء التي تبلغ 24 ساعة. كل رسالة تسويقية، أو رسالة خدماتية (مفيدة)، أو رسالة مصادقة يتم إرسالها أولاً يجب أن تكون موجودة كقالب معتمد.

## نقاط النهاية (Endpoints)

| Method | Endpoint | الغرض |
|--------|----------|---------|
| `POST` | `/{version}/{waba-id}/message_templates` | إنشاء قالب |
| `GET` | `/{version}/{waba-id}/message_templates` | سرد القوالب |
| `GET` | `/{version}/{template-id}` | الحصول على تفاصيل القالب |
| `POST` | `/{version}/{template-id}` | تعديل قالب |
| `DELETE` | `/{version}/{waba-id}/message_templates` | حذف قالب (أو قوالب) |

---

## CREATE — المعلمات (Parameters)

### مطلوبة

| Field | Type | Valid Values | ملاحظات |
|-------|------|-------------|-------|
| `name` | string | `^[a-z0-9_]+$` | بحد أقصى 512 حرفًا. أحرف صغيرة، أرقام، وشرطات سفلية **فقط** |
| `language` | string | `en_US`, `ar`, `en`, ... | رمز اللغة ISO 639 مع إعدادات محلية اختيارية |
| `category` | enum | `MARKETING` \| `UTILITY` \| `AUTHENTICATION` | يتحكم في التسعير والمحتوى المسموح به |

### اختيارية

| Field | Type | Default | ملاحظات |
|-------|------|---------|-------|
| `components` | array | — | الرأس، النص، التذييل، الأزرار |
| `parameter_format` | enum | `POSITIONAL` | `POSITIONAL` (`{{1}}`) أو `NAMED` (`{{name}}`) |
| `allow_category_change` | boolean | `false` | السماح لـ Meta بإعادة تعيين الفئة أثناء المراجعة |
| `message_send_ttl_seconds` | integer | — | قوالب المصادقة فقط |
| `sub_category` | string | — | الفئات الفرعية لقوالب الخدمات |
| `cta_url_link_tracking_opted_out` | boolean | `false` | إلغاء الاشتراك في تتبع الرابط (URL) |
| `library_template_name` | string | — | استنساخ من مكتبة قوالب Meta |

## CREATE — قواعد التحقق (Validation Rules)

| القاعدة | ❌ يتم الرفض | ✅ يتم القبول |
|------|-----------------|-----------------|
| **تنسيق الاسم** | `Hello World`, `hello-world`, `hello#world` | `hello_world`, `order_update_2` |
| **طول الاسم** | أكثر من 512 حرفًا | 512 حرفًا أو أقل |
| **قيمة الفئة** | `TRANSACTIONAL`, `OTP` (قيم قديمة) | `MARKETING`, `UTILITY`, `AUTHENTICATION` |
| **نص متكرر** | نفس نص الرسالة+التذييل مثل قالب موجود في نفس حساب WABA+اللغة | أي نص فريد (تُستثنى قوالب المصادقة `AUTHENTICATION`) |
| **تنسيق المتغير (موضعي)** | `{1}`, `%1`, `[[1]]`, `{{ 1 }}` | `{{1}}`, `{{2}}` |
| **تنسيق المتغير (مسمى)** | `{name}`, `%name%` | `{{customer_name}}` |
| **أحرف المتغير** | `{{name#}}`, `{{price$}}`, `{{id%}}` | `{{name}}`, `{{order_id}}` |
| **تسلسل المتغيرات** | `{{1}}`, `{{2}}`, `{{5}}` (فجوة) | `{{1}}`, `{{2}}`, `{{3}}` |
| **متغيرات معلقة** | بدء/إنهاء النص بـ `{{1}}` فقط | متغير محاط بنص من كلا الجانبين |
| **كثافة المتغيرات** | عدد كبير جداً من `{{n}}` مقارنة بكلمات النص الفعلية | نسبة معقولة من النص إلى المتغيرات |
| **قيم العينة** | قالب بمتغيرات ولكن بدون حقل `example` | `"example": { "body_text": [["John"]] }` |
| **السياسة** | طلب أرقام بطاقات الائتمان، الهويات الوطنية، كلمات المرور | محتوى وظيفي خاص بالعمل |

## المكونات — التفاصيل والحدود (Components)

### HEADER (الرأس - اختياري)

| التنسيق | الحد | ملاحظات |
|--------|-------|-------|
| TEXT | بحد أقصى 60 حرفًا | يدعم **متغير واحد** كحد أقصى. يتطلب عينة. |
| IMAGE | — | يحتاج إلى `header_handle` (يُرفع أولاً) |
| VIDEO | — | يحتاج إلى `header_handle` |
| DOCUMENT | — | يحتاج إلى `header_handle` |
| GIF | بحد أقصى 3.5 MB | في الواقع هو تنسيق MP4 في الخلفية |
| LOCATION | — | فقط في فئات `UTILITY` و `MARKETING` |

### BODY (النص - مطلوب)

| الخاصية | الحد |
|----------|-------|
| الحد الأقصى للأحرف | **1024** |
| المتغيرات | يُسمح بمتغيرات متعددة |
| ماركداون (Markdown) | **غير** مدعوم |
| يتطلب عينة | نعم، عند وجود متغيرات |

### FOOTER (التذييل - اختياري)

| الخاصية | القيمة |
|----------|-------|
| الحد الأقصى للأحرف | **60** |
| المتغيرات | ❌ غير مسموح |
| التنسيق | نص فقط |

### BUTTONS (الأزرار - اختيارية)

| النوع | الحد الأقصى لكل قالب | الحد لكل زر | ملاحظات |
|------|-----------------|-----------------|-------|
| `QUICK_REPLY` | 10 | النص: 25 حرفًا | يجب تجميعها (الكل في البداية أو الكل في النهاية) |
| `URL` | 2 | النص: 25 حرفًا، الرابط: 2000 حرف | متغير واحد في نهاية الرابط (URL) فقط |
| `PHONE_NUMBER` | 1 | النص: 25 حرفًا، الرقم: 20 حرفًا | — |
| `COPY_CODE` | 1 | الرمز: 20 حرفًا | — |

> **⚠️ قاعدة الرؤية على سطح المكتب:** القوالب التي تحتوي على 4 أزرار أو أكثر، أو أزرار الرد السريع المختلطة مع أنواع أخرى، تكون **مخفية على تطبيق WhatsApp لسطح المكتب**. تظهر على الجوال فقط.

> **قاعدة واجهة المستخدم (UI):** عندما يكون إجمالي عدد الأزرار > 3، يظهر أول زرين فقط — ويتم إخفاء الباقي ضمن "رؤية جميع الخيارات".

## EDIT — القواعد (Rules)

```
POST /{version}/{template-id}
```

### ما يمكن تغييره

| Field | الشرط |
|-------|-----------|
| `components` (body, header, buttons) | دائمًا (يُعيد تشغيل المراجعة) |
| `category` | فقط إذا كان القالب `REJECTED` أو `PAUSED` |
| `message_send_ttl_seconds` | في أي حالة |
| `allow_category_change` | في أي حالة |

### ما لا يمكن تغييره (ثابت)

| Field | الحل البديل |
|-------|-----------|
| `name` | حذف + إنشاء جديد |
| `language` | حذف + إنشاء جديد |

### حدود معدل التعديل (Rate Limits)

| حالة القالب | التعديلات المسموح بها |
|---------------|--------------|
| `APPROVED` | بحد أقصى **10 تعديلات كل 30 يومًا**، أو **تعديل واحد كل 24 ساعة** |
| `REJECTED` | غير محدود |
| `PAUSED` | غير محدود |

## DELETE — القواعد (Rules)

```
DELETE /{version}/{waba-id}/message_templates
```

| المعلمة (Param) | التأثير |
|-------|--------|
| `name` فقط | يحذف **جميع** لغات ذلك القالب |
| `name` + `hsm_id` | يحذف لغة **واحدة محددة** |
| `hsm_ids` (مصفوفة) | يحذف ما يصل إلى **100** قالب في المرة الواحدة — إذا كان أي معرّف (ID) غير صالح، **يفشل الطلب بالكامل** |

### قيود الحذف

| الشرط | النتيجة |
|-----------|--------|
| القالب في حالة `DISABLED` | ❌ لا يمكن حذفه |
| القالب كان معتمدًا (`APPROVED`) | يتم **حظر الاسم لمدة 30 يومًا** من إعادة الاستخدام |
| القالب قيد الإرسال عند حذفه | تصبح الحالة `PENDING_DELETION`، ويُعاد محاولة التسليم لمدة 30 يومًا |
| عدم وجود إذن `whatsapp_business_management` | رمز الخطأ `200` |

## مرجع رموز الأخطاء (Error Codes)

### أخطاء الإنشاء / التحقق

| Code | Subcode | الخطأ | ما الذي تسبب فيه | الحل |
|------|---------|-------|-------------------|-----|
| `100` | — | معلمة غير صالحة | الاسم يحتوي على أحرف كبيرة/خاصة، أو فئة غير صالحة | استخدم أسماء بتنسيق `^[a-z0-9_]+$` |
| `100` | `2388024` | اسم مكرر | نفس الاسم+اللغة موجود بالفعل في هذا الـ WABA | استخدم اسمًا مختلفًا |
| `2388040` | — | تم تجاوز الحد الأقصى للأحرف | النص > 1024، الرأس > 60، التذييل > 60 | اختصر الحقل |
| `2388047` | — | تنسيق الرأس غير صحيح | بنية الرأس غير صالحة | تحقق من تنسيق مكون الرأس |
| `2388072` | — | تنسيق النص غير صحيح | صيغة أو بنية المتغير سيئة | تحقق من تنسيق `{{n}}` |
| `2388073` | — | تنسيق التذييل غير صحيح | التذييل يحتوي على متغيرات | قم بإزالة المتغيرات من التذييل |
| `2388293` | — | تم تجاوز نسبة الكلمات/المعلمات | متغيرات كثيرة جداً مقارنة بطول النص | أضف المزيد من النص حول المتغيرات |
| `2388299` | — | معلمة معلقة | يبدأ القالب أو ينتهي بمتغير | أحط المتغير بنص |
| `2388019` | — | تم تجاوز حد القوالب | يحتوي حساب WABA على قوالب أكثر من اللازم | احذف القوالب غير المستخدمة |

### أخطاء وقت الإرسال (عند إرسال رسالة قالب)

| Code | الخطأ | ما الذي تسبب فيه | الحل |
|------|-------|-------------------|-----|
| `132000` | عدم تطابق عدد المعلمات | الطلب يحتوي على عدد متغيرات مختلف عن القالب | طابق المتغيرات بشكل دقيق |
| `132001` | القالب غير موجود أو غير معتمد | اسم (slug) خاطئ، أو لغة خاطئة، أو لا يزال قيد المراجعة | تحقق من الاسم/اللغة/الحالة |
| `132005` | النص المترجم طويل جداً | — | اختصر النص |
| `132007` | انتهاك السياسة | محتوى القالب ينتهك سياسة WhatsApp | راجع محتوى القالب |
| `132012` | تنسيق المتغير غير صحيح | قيم متغيرات مشوهة في طلب الإرسال | أصلح تنسيق المعلمة |
| `132015` | تم إيقاف القالب مؤقتاً | درجة جودة منخفضة | حسّن صلة القالب بالموضوع |
| `132016` | تم تعطيل القالب | تم إيقافه مؤقتاً مرات كثيرة جداً | أنشئ قالبًا جديدًا |
| `132018` | خطأ في التحقق من القالب | المتغيرات في الطلب لا تتطابق مع القالب | تحقق من بنية المعلمة |
| `134101` | القالب لا يزال قيد المزامنة | تم إنشاؤه/تعديله للتو، ولم يتم نشره بعد | انتظر حتى 10 دقائق |
| `134102` | القالب غير متاح (فشلت مزامنة الإعلان) | فشل رابط الإعلان | تحقق من ربط حساب إعلانات Meta |
| `131049` | تم حظر الرسالة (حد لكل مستخدم) | وصل المستخدم إلى حد القوالب التسويقية | طبيعي — حد يفرضه المستخدم |
| `131050` | قام المستخدم بإلغاء الاشتراك | قام المستخدم بإلغاء الاشتراك من الرسائل التسويقية | احترم إلغاء الاشتراك |
| `131063` | تم تعطيل القوالب التسويقية | قام WABA بتعطيل الرسائل التسويقية | تواصل مع دعم Meta |
| `131064` | حد مراسلة الحساب | انتهاكات تصنيف القوالب | راجع فئات القوالب |

## أمثلة التعليمات البرمجية

### ✅ صالح: إنشاء قالب UTILITY

```json
POST /v21.0/{WABA_ID}/message_templates

{
  "name": "order_shipped",
  "language": "en_US",
  "category": "UTILITY",
  "components": [
    {
      "type": "HEADER",
      "format": "TEXT",
      "text": "Order Update",
      "example": { "header_text": ["Order Update"] }
    },
    {
      "type": "BODY",
      "text": "Hi {{1}}, your order {{2}} has shipped and will arrive by {{3}}.",
      "example": {
        "body_text": [["Sarah", "#ORD-1234", "Friday"]]
      }
    },
    {
      "type": "FOOTER",
      "text": "Reply STOP to unsubscribe"
    },
    {
      "type": "BUTTONS",
      "buttons": [
        {
          "type": "URL",
          "text": "Track Order",
          "url": "https://example.com/track/{{1}}",
          "example": ["ABC123"]
        }
      ]
    }
  ]
}
```

### ❌ مرفوض: اسم بأحرف كبيرة

```json
{
  "name": "Order_Shipped",  ← ERROR: uppercase 'O' and 'S'
  "language": "en_US",
  "category": "UTILITY"
}
// Error 100: Parameter name must match ^[a-z0-9_]+$
```

### ❌ مرفوض: النص يبدأ بمتغير (معلق)

```json
{
  "type": "BODY",
  "text": "{{1}} has placed an order."  ← ERROR: starts with variable
}
// Error 2388299: Leading or trailing parameters not allowed
```

### ❌ مرفوض: التذييل يحتوي على متغير

```json
{
  "type": "FOOTER",
  "text": "Order ID: {{1}}"  ← ERROR: variables not allowed in footer
}
// Error 2388073: Message footer format is incorrect
```

### ✅ صالح: تعديل قالب مرفوض (تغيير النص)

```json
POST /v21.0/{TEMPLATE_ID}

{
  "components": [
    {
      "type": "BODY",
      "text": "Hi {{1}}, your updated order is confirmed.",
      "example": {
        "body_text": [["Sarah"]]
      }
    }
  ]
}
```

### ❌ مرفوض: محاولة تغيير اسم القالب (ثابت)

```json
POST /v21.0/{TEMPLATE_ID}

{
  "name": "new_name"  ← ERROR: name is immutable
}
// Field is silently ignored or returns error — name cannot change
```

## SEEN V2 — ملاحظات

| الميزة | كيف يستخدمها SEEN V2 |
|---------|---------------------|
| قوالب WhatsApp | جوهر وحدة البث (Broadcasts) — كافة عمليات الإرسال الجماعي |
| الفئة | `MARKETING` للحملات، `UTILITY` للرسائل الخاصة بالمعاملات |
| خطافات الويب (Webhooks) | `message_template_status_update` → المراقبة من أجل `APPROVED`/`REJECTED` |
| حدود المعدل (Rate limits) | تُطبق لكل WABA — مشتركة بين جميع عملاء SEEN V2 على نفس الـ WABA |
| إلغاء الاشتراك (Opt-out) | يجب التعامل مع `131050` بسلاسة في وحدة البث |

> **ملاحظة ضمان الجودة (QA):** عند اختبار إنشاء قالب، قم دائمًا بتضمين قيم العينة (`example`) لأي قالب يحتوي على متغيرات، وإلا سترفضه واجهة برمجة التطبيقات (API) بخطأ في التحقق.

## GET — سرد القوالب

```http
GET /v23.0/{waba-id}/message_templates
Authorization: Bearer {token}
```

### معلمات الاستعلام (Query Parameters)

| المعلمة (Parameter) | Type | الوصف |
|-----------|------|-------------|
| `fields` | string | حقول مفصولة بفواصل: `name,category,status,language,components` |
| `status` | enum | التصفية حسب: `APPROVED`, `PENDING`, `REJECTED`, `PAUSED`, `DISABLED` |
| `limit` | integer | الحد الأقصى للنتائج لكل صفحة (الافتراضي: 100) |
| `after` | cursor | مؤشر (cursor) الترقيم للصفحة التالية |

### التصفية حسب الحالة — مثال

```bash
# Get only APPROVED templates
GET /v23.0/{waba-id}/message_templates?fields=name,category,status&status=APPROVED
```

```json
{
  "data": [
    { "name": "order_confirmation",  "category": "UTILITY",    "status": "APPROVED", "id": "1387372356726668" },
    { "name": "summer_campaign",     "category": "MARKETING",  "status": "APPROVED", "id": "1304694804498707" }
  ],
  "paging": {
    "cursors": { "before": "QVFIU...", "after": "QVFIU..." },
    "next": "https://graph.facebook.com/v23.0/..."
  }
}
```

### حالات القالب

| الحالة | المعنى |
|--------|---------|
| `APPROVED` | جاهز للاستخدام — يمكن إرساله إلى العملاء |
| `PENDING` | قيد المراجعة بواسطة Meta (عادةً خلال دقائق) |
| `REJECTED` | فشل في المراجعة — راجع سبب الرفض |
| `PAUSED` | تم إيقافه مؤقتاً تلقائياً بسبب تقييم جودة منخفض |
| `DISABLED` | معطل بشكل دائم بعد انخفاض متكرر في الجودة |
| `PENDING_DELETION` | تم طلب الحذف؛ تتم محاولة التسليم لمدة 30 يومًا |
| `LIMIT_EXCEEDED` | وصل حساب WABA إلى حد الـ 250 قالب |

## EDIT — القيود

> **المصدر:** [مستندات Meta — تعديل القوالب](https://developers.facebook.com/documentation/business-messaging/whatsapp/templates/template-management/#limitations)

| القيد | التفاصيل |
|-----------|---------|
| **الحالات المسموح بها** | يمكن فقط تعديل القوالب ذات الحالة `APPROVED`، أو `REJECTED`، أو `PAUSED` |
| **الحقول القابلة للتعديل** | فقط: `category`، `components`، `message_send_ttl_seconds` |
| **لا يمكن عزل المكونات** | يتم استبدال جميع المكونات معًا — يجب عليك إرسال جميع المكونات في طلب التعديل |
| **فئة القوالب المعتمدة** | لا يمكن تغيير فئة قالب في حالة `APPROVED` |
| **حد المعدل (للمعتمدة)** | بحد أقصى **10 تعديلات في 30 يومًا** أو **تعديل واحد في 24 ساعة** |
| **حد المعدل (للمرفوضة/الموقفة مؤقتًا)** | تعديلات غير محدودة |
| **بعد التعديل** | يعود القالب للمراجعة تلقائيًا؛ ويتم اعتماده ما لم يفشل في المراجعة |
| **الاسم ثابت** | لا يمكن تغيير اسم القالب — يتم تجاهل أي حقل `name` في طلب التعديل |

### ✅ مثال قابل للتعديل — تعديل المكونات

```bash
POST /v23.0/{template-id}
Content-Type: application/json
Authorization: Bearer {token}

{
  "components": [
    {
      "type": "HEADER",
      "format": "TEXT",
      "text": "Our {{1}} is on!",
      "example": { "header_text": ["Spring Sale"] }
    },
    {
      "type": "BODY",
      "text": "Shop now through {{1}} and use code {{2}} to get {{3}} off.",
      "example": { "body_text": [["the end of April", "25OFF", "25%"]] }
    }
  ]
}
```

### ✅ تعديل الفئة فقط

```bash
POST /v23.0/{template-id}

{ "category": "MARKETING" }
```

الاستجابة:
```json
{ "success": true }
```

## DELETE — القيود والطرق

> **المصدر:** [مستندات Meta — حذف القوالب](https://developers.facebook.com/documentation/business-messaging/whatsapp/templates/template-management/#delete-templates)

| القيد | التفاصيل |
|-----------|---------|
| **تم الإرسال ولكن لم يتم التسليم** | تصبح الحالة `PENDING_DELETION` — ويُعاد محاولة التسليم لمدة 30 يومًا |
| **حظر لمدة 30 يومًا** | بعد حذف قالب `APPROVED`، لا يمكن إنشاء قالب جديد يحمل **نفس الاسم لمدة 30 يومًا** |
| **لا يمكن حذف المعطلة** | لا يمكن حذف القوالب ذات الحالة `DISABLED` |
| **إذن مطلوب** | إذن `whatsapp_business_management` على رمز الوصول (access token) |

### الطريقة 1 — الحذف بالاسم (جميع اللغات)

```bash
DELETE /v23.0/{waba-id}/message_templates?name=order_confirmation
Authorization: Bearer {token}
```

⚠️ يحذف **جميع إصدارات اللغات** لاسم القالب هذا.

### الطريقة 2 — الحذف بالمعرف ID (لغة محددة فقط)

```bash
DELETE /v23.0/{waba-id}/message_templates?hsm_id=1407680676729941&name=order_confirmation
Authorization: Bearer {token}
```

### الطريقة 3 — حذف متعدد بواسطة المعرفات IDs (بحد أقصى 100 لكل طلب)

```bash
DELETE /v23.0/{waba-id}/message_templates?hsm_ids=[1387372356726668,1304694804498707]
Authorization: Bearer {token}
```

الاستجابة لجميع طرق الحذف:
```json
{ "success": true }
```

## الإيقاف المؤقت للقوالب والجودة

يتم إيقاف القوالب مؤقتًا بشكل تلقائي عندما تصل إلى **جودة منخفضة** (`quality_score: RED`).

### جدول مدة الإيقاف المؤقت

| الحالة (Instance) | مدة الإيقاف المؤقت |
|----------|---------------|
| المرة الأولى | 3 ساعات |
| المرة الثانية | 6 ساعات |
| المرة الثالثة | **معطل بشكل دائم** |

### إشعارات الإيقاف المؤقت

يتم إشعارك عبر:
- إشعارات مدير واتساب (WhatsApp Manager)
- البريد الإلكتروني
- خطاف الويب `message_template_status_update`

### حمولة خطاف الويب (تغيير الحالة - Payload)

```json
{
  "entry": [{
    "changes": [{
      "field": "message_template_status_update",
      "value": {
        "message_template_id": 1234567890,
        "message_template_name": "your_template",
        "message_template_language": "en_US",
        "event": "PAUSED",
        "reason": "LOW_QUALITY"
      }
    }]
  }]
}
```

### إلغاء الإيقاف المؤقت

| الطريقة | الكيفية |
|--------|-----|
| **تلقائي** | يتم إلغاء الإيقاف المؤقت للقالب تلقائيًا بعد انتهاء مدة الإيقاف |
| **يدوي (API)** | `POST /{template-id}/unpause` |
| **يدوي (واجهة المستخدم)** | انقر على "manually unpause it" (إلغاء الإيقاف المؤقت يدويًا) في WhatsApp Manager |

> بعد الإلغاء التلقائي للإيقاف المؤقت، تتم **إعادة تعيين** تقييم الجودة بناءً على أحدث ملاحظات العملاء.

### تقديم التماس (قالب مرفوض/معطل)

1. اذهب إلى **WhatsApp Manager → Account Tools → Message Templates**
2. حدد القالب → Edit (تعديل) → انقر على **Add Sample** (إضافة عينة)
3. قم بتعبئة قيم متغيرات نموذجية وواقعية
4. أرسل للمراجعة — القرارات تُتخذ عادةً خلال **24 ساعة**

| Error Code | Subcode | الرسالة | الحل |
|-----------|---------|---------|-----|

## تحليلات القالب (Template Analytics)

تعرض التحليلات مقاييس مثل: عدد الرسائل **المرسلة (sent)**، **التي تم تسليمها (delivered)**، **المقروءة (read)**، و**النقرات على الأزرار (button click)** لكل قالب.

### تفعيل التحليلات

```bash
POST /{waba-id}?is_enabled_for_insights=true
Authorization: Bearer {token}
```

### الحصول على التحليلات

```bash
GET /v23.0/{waba-id}/template_analytics
  ?start=1718064000
  &end=1718150400
  &granularity=DAILY
  &template_ids=[1421988012088524,2632273056924580]
  &metric_types=SENT,DELIVERED,READ,CLICKED,COST
```

### معلمات التحليلات

| المعلمة (Parameter) | مطلوب | Type | الوصف |
|-----------|----------|------|-------------|
| `start` | ✅ | طابع زمني (UNIX timestamp) | بداية النطاق الزمني |
| `end` | ✅ | طابع زمني (UNIX timestamp) | نهاية النطاق الزمني |
| `granularity` | ✅ | enum | يجب أن تكون `DAILY` |
| `template_ids` | ✅ | array | مصفوفة من معرفات القوالب (IDs) — **بحد أقصى 10** |
| `metric_types` | ❌ | array | `SENT`, `DELIVERED`, `READ`, `CLICKED`, `COST` |

### مثال على استجابة التحليلات

```json
{
  "data": [{
    "granularity": "DAILY",
    "product_type": "cloud_api",
    "data_points": [{
      "template_id": "2632273056924580",
      "start": 1718064000,
      "end": 1718150400,
      "sent": 1500,
      "delivered": 1480,
      "read": 1200,
      "clicked": [
        { "type": "quick_reply_button", "button_content": "Order Now", "count": 540 },
        { "type": "url_button",         "button_content": "See Deals",  "count": 210 }
      ],
      "cost": [
        { "type": "amount_spent", "value": 45.30 }
      ]
    }]
  }]
}
```

### قيود التحليلات

| القيد | التفاصيل |
|-----------|---------|
| **الاحتفاظ بالبيانات** | بيانات القراءة والنقر متاحة فقط لمدة **7 أيام** من تاريخ الإرسال؛ بعد ذلك، يتم تصفير العدادات |
| **نقرات الأزرار** | متاحة فقط لقوالب `MARKETING` و `UTILITY` |
| **المناطق المستثناة** | حسابات WABA في الاتحاد الأوروبي، واليابان، أو التي تمتلك أرقام هواتف من هذه المناطق **غير مدعومة** |
| **الحد الأقصى للقوالب لكل طلب** | 10 معرفات (IDs) قوالب لكل استدعاء للتحليلات |

### إلغاء الاشتراك في تتبع النقرات (لكل قالب)

```bash
POST /{template-id}?cta_url_link_tracking_opted_out=true&category=MARKETING
```

## الأرشفة والحذف التلقائي

القوالب غير النشطة لمدة **12+ شهرًا** تتم أرشفتها تلقائيًا وتُجدول **للحذف بعد 28 يومًا**.

- يمكنك أرشفة القوالب أو إلغاء أرشفتها يدويًا بشكل جماعي عبر واجهة برمجة التطبيقات (API)
- تُحسب القوالب المؤرشفة ضمن حد الـ 250 قالبًا حتى يتم حذفها
