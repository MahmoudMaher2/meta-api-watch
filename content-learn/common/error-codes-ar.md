# رموز الخطأ — المرجع الشامل

تتبع جميع أخطاء WhatsApp Business API هيكلًا متسقًا. فهم هذه الرموز يمنع إضاعة الوقت في تصحيح الأخطاء.

## هيكل استجابة الخطأ

```json
{
  "error": {
    "message": "Invalid parameter",
    "type": "OAuthException",
    "code": 100,
    "error_subcode": 2388024,
    "error_data": {
      "messaging_product": "whatsapp",
      "details": "Template name already exists"
    },
    "fbtrace_id": "AbCd1234..."
  }
}
```

| الحقل | المعنى |
|-------|---------|
| `code` | فئة الخطأ الرئيسية (Top-level) |
| `error_subcode` | خطأ فرعي محدد ضمن الفئة |
| `error_data.details` | شرح مقروء للإنسان |
| `fbtrace_id` | استخدم هذا المعرف عند التواصل مع دعم Meta |

## رموز الخطأ الرئيسية

### أخطاء Graph API العامة

| الرمز | الاسم | السبب الشائع |
|------|------|-------------|
| `1` | خطأ غير معروف | مشكلة مؤقتة في Meta — أعد المحاولة |
| `2` | الخدمة غير متاحة مؤقتًا | انقطاع في Meta — أعد المحاولة مع تأخير زمني (backoff) |
| `4` | تم الوصول إلى الحد الأقصى لطلبات التطبيق | تم تجاوز حد المعدل (Rate limit) |
| `10` | تم رفض الإذن | يفتقر التطبيق إلى الإذن المطلوب |
| `100` | معلمة غير صالحة | تنسيق قيمة خاطئ أو حقل مفقود |
| `190` | رمز وصول غير صالح/منتهي الصلاحية | انتهت صلاحية الرمز (Token) أو تم إلغاؤه |
| `200` | يتطلب إذنًا | إذن Facebook/Meta مفقود |

## رموز الخطأ الخاصة بـ WhatsApp (13xxxx)

### 130xxx — التقييد وحدود المعدل (Throttling & Rate Limits)

| الرمز | الخطأ | السبب | الحل |
|------|-------|-------|-----|
| `130429` | تم الوصول إلى حد المعدل | إرسال أكثر من 80 رسالة/ثانية | خفض المعدل إلى 80/ثانية |
| `130472` | تم الوصول إلى حد رسائل المستخدم | حد المستخدم الفردي | قلل الإرسال (Back off) لهذا المستخدم |

### 131xxx — أخطاء إرسال الرسائل

| الرمز | الخطأ | السبب | الحل |
|------|-------|-------|-----|
| `131000` | فشل الإرسال | خطأ عام | تحقق من `details` لمعرفة التفاصيل |
| `131005` | تم رفض الإذن | إذن `whatsapp_business_messaging` مفقود | امنح الإذن في مراجعة التطبيق |
| `131008` | معلمة مطلوبة مفقودة | فقدان `to`، `type`، إلخ. | أضف الحقل المفقود |
| `131009` | معلمة غير صالحة | رقم هاتف خاطئ، وسائط غير صالحة | تحقق من الصحة قبل الإرسال |
| `131016` | الخدمة غير متاحة | WhatsApp Cloud API معطل | أعد المحاولة مع تأخير زمني مضاعف (exponential backoff) |
| `131021` | المستلم غير صالح | الرقم غير مسجل في WhatsApp | تحقق من الرقم أولاً |
| `131026` | تعذر تسليم الرسالة | المستخدم لا يمكن الوصول إليه أو قام بالحظر | قم بتسجيل الخطأ وتخطاه |
| `131042` | أهلية النشاط التجاري — المدفوعات | غير مفعل للمدفوعات | فعّل المدفوعات لـ WABA |

| `131047` | إعادة التفاعل مطلوبة | لا توجد نافذة 24 ساعة مفتوحة | أرسل قالبًا (template) |
| `131048` | حد معدل البريد العشوائي (Spam) | رسائل كثيرة جدًا للمستخدم | قيد الإرسال لكل مستخدم (Throttle) |
| `131049` | حد قوالب التسويق | وصل المستخدم للحد المفروض من Meta | أمر طبيعي — التزم بالحد |

| `131052` | خطأ في تنزيل الوسائط | الرابط لا يمكن الوصول إليه أو منتهي الصلاحية | أعد رفع الوسائط |
| `131053` | خطأ في رفع الوسائط | فشل الرفع | أعد محاولة الرفع |
| `131057` | الحساب قيد الصيانة | WABA قيد الصيانة | انتظر وأعد المحاولة |

## استراتيجية معالجة الأخطاء لـ SEEN V2

| رمز الخطأ | الإجراء |
|------------|--------|
| `131000`, `131016`, `2` | أعد المحاولة مع تأخير زمني مضاعف (بحد أقصى 3 محاولات) |
| `131026` | قم بتسجيل الخطأ وتخطاه — لا تعد المحاولة |
| `131047` | ضع الرسالة في قائمة الانتظار على أنها "تتطلب قالبًا" (template required) |
| `131048`, `130429` | قلل الإرسال (Back off) — انتظر قبل إرسال المزيد |
| `131049`, `131050` | أزل المستخدم من قائمة البث (broadcast list) |
| `131052`, `131053` | أعد رفع الوسائط وأعد المحاولة |
| `132001`, `132006`, `132016` | نبه فريق ضمان الجودة (QA) — مشكلة في القالب |
| `132000` | خطأ برمجي (Bug) — أصلح عدد المعلمات في الكود |
| `190` | قم بتحديث رمز الوصول (access token) |
| `4`, `10` | نبه المطور — مشكلة في الأذونات |\n\n

**أخطاء العميل مقابل أخطاء الخادم**
- **أخطاء العميل (4xx):** تحدث عادة بسبب إدخال غير صالح، أو تجاوز حدود الإرسال، أو انتهاء صلاحية التوكن. يجب إصلاح الطلب قبل إعادة المحاولة.
- **أخطاء الخادم (5xx):** تحدث بسبب مشاكل مؤقتة من طرف Meta. يجب إعادة المحاولة لاحقاً مع استخدام التراجع التدريجي (Exponential Backoff).
