Error Codes — Complete Reference

All WhatsApp Business API errors follow a consistent structure. Understanding the codes prevents wasted debugging time.

Error Response Structure

{

  "error": {


    "message": "Invalid parameter",


    "type": "OAuthException",


    "code": 100,


    "error_subcode": 2388024,


    "error_data": {


      "messaging_product": "whatsapp",


      "details": "Template name already exists"


    },


    "fbtrace_id": "AbCd1234..."


  }


}

FieldMeaning
codeTop-level error category
error_subcodeSpecific sub-error within category
error_data.detailsHuman-readable explanation
fbtrace_idUse this when contacting Meta Support

تتبع جميع أخطاء WhatsApp Business API هيكلًا متسقًا. فهم هذه الرموز يمنع إضاعة الوقت في تصحيح الأخطاء.

  "error": {


    "message": "Invalid parameter",


    "type": "OAuthException",


    "code": 100,


    "error_subcode": 2388024,


    "error_data": {


      "messaging_product": "whatsapp",


      "details": "Template name already exists"


    },


    "fbtrace_id": "AbCd1234..."


  }


}

الحقلالمعنى

fbtrace_idاستخدم هذا المعرف عند التواصل مع دعم Meta
